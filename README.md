# ALLOY COMPOSITION OPTIMIZATION

A model that predicts best alloy composition for desired values of Electrical conductivity, Hardness, Tensile strength, Yield Strength

Dataset is of Copper based alloys

##It has 2 Model :## 1. NLP based input values of desired values
                     2. Numeric min. & max. desired values

Here used is RandomForest and GradientForest Regression
